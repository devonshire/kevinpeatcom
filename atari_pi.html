Atari Pi Project

Components | Arduino Wiring | Arduino Coding | Pi Mounting | Pi Software | Final Assembly

My first computers were from Atari, initially a 400 with the hard membrane keyboard (ugh!) and then an 800XL and finally a couple of ST's. I still enjoy messing around with Atari kit but the hardware is getting old now and the lack of modern features like HDMI and USB can be a pain.

I saw a project where someone had built a Raspberry Pi into an old Sinclair Spectrum case and I thought I would do something similar and build a Raspberry Pi into an Atari ST case to use with an ST emulator:

I have updated this project here to use a Raspberry Pi 3.
